Monster Hunter X/Gen Reclaim Item Packs

Game type is detected automatically based on save size.
MHGen ≈ 3.81 MB
MHX ≈ 3.53 MB
DO NOT USE THIS TOOL IF SWITCHING REGIONS LIKE EU TO USA (or backwards), TW TO JP (or backwards). YOU WILL LOSE YOUR CHARACTER'S ONLY CHANCE TO CLAIM DIFFERENT ITEM PACKS FROM OTHER REGIONS!
Basically: the only valid use of this tool is when you migrate your character from a game WITH DIFFERENT NAME (and item packs).
This tool replaces specific bytes with 0x00 to reclaim item packs.
The reason of it's existence is to help people get item packs again when porting their saves from generations to cross, or backwards.
If by chance you mess up after reading all the warnings, you have to read the source code of this tool and understand how to fix your mistake by yourself :)

New tracking system:
- If tracking byte doesn't exist, the save file will be modified.
- Each slot can be claimed only once across all tool uses.
A patched copy of the save file will automatically download (original save given also saved as "system_backup").
